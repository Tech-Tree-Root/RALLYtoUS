蕭光宇（しょうこうう）（デューク大学、2009年クラス）は、欧州委員会の #EUvsVirus CTO および米国の国立学芸員を務めています。


HackDuke を設立めた後，彼は世界の専門家になった。 HackDuke は、社会的影響とデューク大学の旗艦イベントの最高のハッカソンです。


2017 年 1 月，彼は Tech Tree Root を設立しました。 Tech Tree Root は、 EUvsVirus 創設者であり、米国の国家ハッカソン組織です。


オンライン連絡  >
Tech Tree Root は、技術および起業家教育の機会とリソースを生み出すことにより、イノベーションエコシステムを作成します

18か月以内に, 彼は テキサス州、オクラホマ州、アーカンソー州のすべての主要な学校にハッカソンを立ち上げ 、オーガナイザーをトレーニングすることにより、米国中南部のイノベーション砂漠を変革しました。

彼は25,000人以上の学生を教育し、受賞者を指導しました。 Tech Tree Root はカリフォルニアとアイスランドにまで事業を拡大し、その後コロナウイルスに対するアメリカの取り組みを主導しました。

HackCOVID は、差し迫った混乱の中で皆に希望と目的の感覚をもたらすために、アメリカのコロナウイルス国家緊急宣言の前に始まりました。


蕭光宇は才能のあるデザイナーとエンジニアを募集しました。毎週、彼らはテクノロジーを使用してコロナウイルスと戦った。 HackCOVID は米国の全国ハッカソンになりました。


オンライン連絡  >
4つの週末が成功した後、欧州委員会は蕭光宇に #EUvsVirus のサポートを要請しました。蕭光宇は、ミッション主導のイニシアチブ（ハッカソン + マッチソン + EIC COVID プラットフォーム）の開始を支援しました。
チームは 30,000 人以上の参加者と 2,164 の国際チームで世界記録を樹立しました。次に、専門家の審査員が 120 の優勝ソリューションを選択しました。

その後、Mathathonで、＃EUvsVirusチームは勝者と500以上の公的および私的エンティティを組み合わせて、世界記録の2,235の新しいパートナーシップを構築しました。多くの命を救う革新が生産を開始しました。

今、ジョーは私たちの未来を救うために時差と言語の壁を克服しています。一緒に働き、私たちの未来を救いましょう！
